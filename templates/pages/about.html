Sobre o Tech Hub

Marketplace de componentes e produtos de tecnologia, com foco em compra, venda e troca entre pessoas e lojas.

info O projeto

O Tech Hub é um projeto desenvolvido para a faculdade, com o objetivo de aplicar conceitos de desenvolvimento web full-stack: autenticação e perfis (PF e lojas), CRUD de anúncios, carrinho, checkout simulado, um fluxo completo de troca com negociação, notificações, controle de estoque, e um painel financeiro.

O pagamento é simulado (QR Code ilustrativo) — nenhuma transação real é processada. O diferencial do projeto é o sistema de trocas entre usuários.
